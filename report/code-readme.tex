در پیوست، یک فایل فشرده قرار دارد که کد مربوط به این پروژه در آن موجود است. جدیدترین نسخهٔ این کد در مخزن آنلاین گیتهاب ذیل آدرس \url{https://github.com/m2-farzan/BCI-Project} نگهداری می‌شود.

فایل اصلی، فایل \ورب{main.py} است که با اجرای آن، تمام نمودارهای گزارش تولید و به صورت فایل‌های \ورب{png} در کنار فایل \ورب{main.py} ذخیره می‌شوند. اجرای این فایل به دو روش ممکن است:
\شروع{فقرات}
\فقره \متن‌سیاه{روش سریع (توصیه شده):} در این روش، نیازی به دانلود و نصب جداگانهٔ پایتون و کتابخانه‌های پیش نیاز نیست و به جای آن، از یک نسخهٔ قابل حمل پایتون که نسخهٔ صحیح کتابخانه‌های مورد نیاز به طور آماده در کنار آن قرار داده شده است، استفاده می‌شود. برای این کار کافی است فایل موجود در آدرس \url{https://mfarzan.ir/bci-project/portable} دانلود شود.\پانوشت{به دلیل اینکه نسخهٔ باینری کتابخانه‌های متعددی در این فایل قرار دارد، حجم آن ممکن است چند صد مگابایت باشد.} پس از دانلود فایل و خارج کردن آن از حالت فشرده، مشاهده می‌شود که در پوشهٔ ایجاد شده یک فایل \ورب{main.bat} قرار دارد. با کلیک روی این فایل، کد اجرا می‌شود.\پانوشت{اجرای کد بسته به سیستم میزبان ممکن است تا ۳ ساعت طول بکشد.} این فایل به طور درونی، فایل \ورب{main.py} را با استفاده از فایل اجرایی پایتون موجود در پوشهٔ \ورب{python}، راه‌اندازی می‌کند. این روش فقط برای سیستم عامل ویندوز ۱۰ با معماری پردازندهٔ \مل{x86\_64} تست شده است.
\فقره \متن‌سیاه{روش دستی:} در این روش، ابتدا باید نرم‌افزار پایتون ۳ روی سیستم نصب باشد. سپس لازم است با استفاده از دستور زیر، کتابخانه‌های پیش‌نیاز نصب شوند:
\begin{Verbatim}
python3 -m pip install -r requirements.txt
\end{Verbatim}
همچنین لازم است کتابخانهٔ \مل{MOABB} به طور دستی دانلود و نصب شود. برای این کار لازم است  ابتدا کتابخانه از آدرس \url{https://github.com/m2-farzan/moabb} دانلود شود و سپس در مسیر ایجاد شده، دو دستور زیر اجرا شود:
\begin{Verbatim}
python3 -m pip install -r requirements.txt
python3 setup.py develop
\end{Verbatim}
